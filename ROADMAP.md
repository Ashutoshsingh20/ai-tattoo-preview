# Product Roadmap

## Phase 1: MVP ✅ (Week 1)
- [x] Core camera capture
- [x] Basic tattoo overlay
- [x] Drag & resize controls
- [x] Save preview
- [x] 8 sample designs
- [x] GitHub repo + README

## Phase 2: Real Designs (Week 2-3)
- [ ] Replace emojis with 50+ real tattoo designs
- [ ] Organize by category (Traditional, Tribal, Watercolor, Minimalist, etc.)
- [ ] AI-generated designs (DALL-E, Midjourney, or Stable Diffusion)
- [ ] Design upload feature (user-provided artwork)
- [ ] Design favoriting/collections

## Phase 3: Enhanced AR (Week 4-5)
- [ ] Body part detection (MediaPipe Pose)
- [ ] Auto-placement suggestions
- [ ] Rotation controls
- [ ] Opacity/transparency slider
- [ ] Multiple tattoos at once
- [ ] Skin tone adaptation

## Phase 4: Social & Sharing (Week 6)
- [ ] Instagram Stories integration
- [ ] TikTok share button
- [ ] WhatsApp preview sharing
- [ ] Gallery of saved previews
- [ ] Community showcase (optional consent)
- [ ] Viral loop: "Made with AI Tattoo Preview" watermark

## Phase 5: Monetization (Week 7-8)
- [ ] User accounts (email + social login)
- [ ] Freemium: 3 free previews → $4.99/month
- [ ] Pay-per-preview: $0.99/download
- [ ] Stripe integration
- [ ] Subscription management
- [ ] Artist referral tracking

## Phase 6: Artist Marketplace (Month 3)
- [ ] Artist profiles
- [ ] Custom design commissions
- [ ] Booking integration (Calendly, Acuity)
- [ ] Review system
- [ ] Geolocation-based artist finder
- [ ] Affiliate commission (10-15% per booking)

## Phase 7: Advanced Features (Month 4+)
- [ ] AI design generator (prompt → tattoo)
- [ ] Tattoo aging simulation (show how it looks in 5/10 years)
- [ ] Pain level heatmap (based on body placement)
- [ ] Cost estimator (size + complexity + location)
- [ ] Aftercare instructions
- [ ] Tattoo removal cost calculator

## Phase 8: B2B (Month 6+)
- [ ] White-label for tattoo studios ($99-$299/month)
- [ ] Studio dashboard (customer previews + booking)
- [ ] Embedded widget for studio websites
- [ ] Custom branding
- [ ] Analytics dashboard

## Growth Metrics to Track

**Week 1-4 (MVP Validation)**:
- Unique visitors
- Preview completion rate
- Share rate
- Time on site

**Month 2-3 (Product-Market Fit)**:
- Free → Paid conversion
- Churn rate
- Avg. previews per user
- Artist finder CTR

**Month 4+ (Growth)**:
- MRR (Monthly Recurring Revenue)
- CAC (Customer Acquisition Cost)
- LTV (Lifetime Value)
- Viral coefficient (invites/shares)

## Success Criteria

**Milestone 1**: 1,000 previews created
**Milestone 2**: 100 paid users
**Milestone 3**: $1K MRR
**Milestone 4**: 5 artist partnerships
**Milestone 5**: $10K MRR (sustainable)

## Potential Pivots

If traction is slow:
1. **Niche down**: Focus on specific style (e.g., minimalist tattoos only)
2. **Free forever**: Monetize via artist referrals only
3. **B2B first**: Sell to studios before consumers
4. **AI generator focus**: Pivot to AI tattoo design tool
